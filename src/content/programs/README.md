# Loan Program Content

Add loan program pages as `.mdx` files in this directory.

## File format

Each file should include YAML frontmatter:

```mdx
---
title: "FHA Loans"
description: "Learn about FHA loan requirements and benefits."
icon: "home"
---

Program details in MDX.
```

## Naming convention

Use kebab-case matching the desired URL slug:

```
fha-loans.mdx  →  /programs/fha-loans
```
