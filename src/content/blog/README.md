# Blog Content

Add blog posts as `.mdx` files in this directory.

## File format

Each file should include YAML frontmatter:

```mdx
---
title: "Your Post Title"
description: "A short summary for SEO and previews."
date: "2026-01-15"
author: "Author Name"
tags: ["mortgage", "tips"]
---

Post body in MDX (Markdown + JSX components).
```

## Naming convention

Use kebab-case matching the desired URL slug:

```
first-time-homebuyer-guide.mdx  →  /blog/first-time-homebuyer-guide
```
